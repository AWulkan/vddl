## Installation
```
npm install --save vddl
```

or

```
yarn add vddl
```

## Browser Support

Modern browsers and Internet Explorer 10+.

## Development

```js
npm install
npm run watch

// build
npm run build

// run example
cd example && npm install
npm run dev
```
